Структура папки устроена таким образом.

- MODELS
-- Category 1
 -- Char1
   -- *.pth
   -- *.index ( optional )
   -- *.jpg or *.png ( optional)
   -- *.txt ( name, description, ( male || female || child )
 -- Char2
   -- *.pth
   -- *.index ( optional )
   -- *.jpg or *.png ( optional)
   -- *.txt ( name, description, ( male || female || child )
-Category 2
 -- default.jpg
 -- script.py
 ***
 
 
в *.txt пишите отдельна каждой строчке. ( Имя, описание, пол (female, male , child )  ) Пример:

alica.txt:
Alica
